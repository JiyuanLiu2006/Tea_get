# 高校教师数据抓取程序
一个基于**Python**的高校教师信息自动采集工具，支持从高校官网教师列表页自动抓取教师姓名、职称、邮箱、个人简介等信息，并导出为 CSV 文件。
### 分为终端版（Terminal）和图形化版（GUI）两个版本
Terminal版在`Tea_Get_Terminal`文件夹中，其所有配置集中在 `Config.py` 文件，打开后修改文件中的配置参数。
GUI版在`Tea_Get_GUI`文件夹中，有图形化界面，运行后在 **采集配置** 面板中填写以下信息，在 **过滤排除词** 面板中，用顿号（`、`）分隔不需要抓取的关键词。
具体操作和注意事项可查看文件夹中各自的README文件。
### 本项目仅供学习与研究使用。使用者应仅用于合法用途，如学术研究、公开信息收集等。请遵守目标网站的 `robots.txt` 和使用条款。本工具仅采集已公开的教师信息，不涉及任何非公开数据。
